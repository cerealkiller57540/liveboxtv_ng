# Orange Livebox TV UHD

Pilotez vos décodeurs **Orange Livebox TV UHD** depuis Home Assistant.

- Découverte automatique (SSDP + DHCP), multi-décodeurs
- Volume / mute réels via UPnP, zapping, télécommande
- Liste des chaînes + EPG (programme en cours) dynamiques depuis Orange
- Robuste : décodeur en veille = `unavailable` (jamais perdu), IP DHCP suivie automatiquement

Configuration 100 % par l'interface. Voir le [README](https://github.com/cerealkiller57540/liveboxtv_ng).
